# Plateforme-num-rique-de-gestion-d-tabliissemets-scolaire
Sahel Digital Systems développe une plateforme numérique moderne de gestion scolaire basée sur une architecture microservices. Elle intégrera la gestion des élèves, enseignants, notes, présences, emplois du temps, inscriptions et services administratifs dans un environnement sécurisé, évolutif et performant.
